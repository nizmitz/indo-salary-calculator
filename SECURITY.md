# Security Policy

## Supported Versions

The following versions of this project are currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 0.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it immediately.

You can report vulnerabilities by:

1. Opening a GitHub Issue (if it is not a sensitive/exploitable bug).
2. Contacting the maintainer directly via GitHub or the contact information provided at [nizmitz.com](https://nizmitz.com).

Please do not disclose security vulnerabilities publicly until they have been addressed. We appreciate your help in keeping this project secure.
